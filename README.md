* CF_with_kNN
  * CF알고리즘을 구현한 코드.

* EDA
  * MovieLens data를 간단하게 탐색.

* movielens
  * movielens 데이터의 1M, 100K 버전이 담긴 데이터 폴더.

* similarity_measure
  * 다양한 유사도 알고리즘이 담긴 모듈.
